# 📊 Customer Data Analysis (SQL)

---

## 📊 Project Overview
This project analyzes customer orders and product sales from an e-commerce database using SQL. The goal is to extract meaningful insights related to customer behavior, revenue generation, and product performance.

---

## 🎯 Objectives
- Analyze customer spending patterns  
- Evaluate product performance  
- Track order frequency and trends  
- Identify high-value customers  
- Understand regional sales distribution  

---

## ⚙️ Key Analysis Performed
- Total revenue and customer-wise spending  
- Order frequency and repeat customer analysis  
- Product-wise sales and top-performing items  
- Region-wise sales distribution  
- Time-based order trends (monthly/annual)  

---

## 🛠️ SQL Techniques Used
- Joins (Inner Join, Left Join)  
- Aggregations (SUM, COUNT, AVG)  
- Group By & Order By  
- Window Functions  
- Common Table Expressions (CTE)  
- Subqueries  

---

## 📈 Key Insights
- Identified top customers contributing maximum revenue  
- Highlighted best-selling products  
- Observed patterns in repeat purchases  
- Analyzed regional differences in sales performance  
- Tracked sales trends over time  

---

## 🚀 Project Outcome
Transformed raw e-commerce data into **structured insights** that can support better decision-making in marketing, sales strategy, and customer engagement.
